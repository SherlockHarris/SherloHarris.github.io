ECS安装JDK

Posted on 2020-02-16

官网下载JDK

JDK下载地址官网

将jdk安装到指定目录（可以自定义）

上传到服务器 /usr/local下

这里利用 SSH Secure File Transfer Client 上传

解压

tar -zxvf jdk-8u241-linux-x64.tar.gz

这里就可以把压缩包删掉了

改名（方便书写和更改环境变量）

mv jdk1.8.0_241/ jdk8

写入环境变量

命令： 

vim /etc/profile

按 i 进入编辑模式，输入下面内容

JAVA_HOME=/usr/local/jdk8
CLASSPATH=$JAVA_HOME/lib/
PATH=$PATH:$JAVA_HOME/bin
export PATH JAVA_HOME CLASSPATH

按 shift+ ：键，输入wq退出文档

重启配置文档

source /etc/profile

检查是否安装完毕

java -version
